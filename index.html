Yum-alert

Email alert for your yum updates !

Download .zip Download .tar.gz View on GitHub

Presentation

Yum-Alert is a simple bash script to monitor your yum updates, and send you an email if news packages are available !

Works (and tested) with CentOS Linux...

Email example

Server : MyGreatServer --> News packages available !

Installation and configuration

The script must be executable, that's all :)

Put this script in /etc/cron.weekly/ or /etc/cron.daily/
